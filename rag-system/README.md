# RAG System — FastAPI + FAISS + SentenceTransformers + Ollama/Mistral

A production-quality Retrieval-Augmented Generation system that lets you upload documents, index them with FAISS, and ask natural-language questions answered by a local Mistral LLM via Ollama.

## Features

- 📄 **Document Upload** — Upload `.txt`, `.md`, and `.pdf` files
- 🔍 **Semantic Search** — FAISS vector similarity search with SentenceTransformers embeddings
- 🤖 **Local LLM** — Answers generated by Mistral via Ollama (runs entirely on your machine)
- 🌊 **Streaming Responses** — Token-by-token streaming to the UI via Server-Sent Events
- 📎 **Source Citations** — See exactly which document chunks were used for each answer
- 💾 **Persistent Index** — FAISS index is saved to disk and survives server restarts
- 🎨 **Premium UI** — Dark glassmorphism design with smooth animations

## Prerequisites

1. **Python 3.10+**
2. **Ollama** — Install from [ollama.com](https://ollama.com)
3. **Mistral model** — Pull it after installing Ollama:
   ```bash
   ollama pull mistral
   ```

## Quick Start

### 1. Clone & navigate
```bash
cd rag-system
```

### 2. Create a virtual environment
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Configure (optional)
```bash
copy .env.example .env
# Edit .env to customise model, chunk size, etc.
```

### 5. Start Ollama (in a separate terminal)
```bash
ollama serve
```

### 6. Run the server
```bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 7. Open in browser
Navigate to **http://localhost:8000**

## API Endpoints

| Method | Endpoint          | Description                    |
|--------|-------------------|--------------------------------|
| GET    | `/api/health`     | Health check & system info     |
| POST   | `/api/documents/` | Upload and ingest a document   |
| GET    | `/api/documents/` | List indexed documents         |
| POST   | `/api/query/`     | Ask a question (non-streaming) |
| POST   | `/api/query/stream` | Ask a question (SSE streaming) |

## Configuration

| Variable        | Default             | Description                         |
|-----------------|---------------------|-------------------------------------|
| EMBEDDING_MODEL | all-MiniLM-L6-v2    | SentenceTransformer model name      |
| OLLAMA_BASE_URL | http://localhost:11434 | Ollama server URL                |
| OLLAMA_MODEL    | mistral             | LLM model name                      |
| CHUNK_SIZE      | 500                 | Characters per chunk                |
| CHUNK_OVERLAP   | 50                  | Overlap between chunks              |
| TOP_K           | 5                   | Number of chunks to retrieve        |
